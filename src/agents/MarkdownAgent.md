# MarkdownAgent — Renderização de Conteúdo

Responsável pela transformação de texto Markdown brutos em uma UI rica e legível.

## Responsabilidades
- Renderizar `.md` usando `react-markdown`.
- Configurar suporte para GFM (GitHub Flavored Markdown) via `remark-gfm`.
- Estilizar headings, tabelas, blocos de código e links.
- Preparar integração com `prismjs` para Realce de Sintaxe (syntax highlighting).
- Otimizar acessibilidade e legibilidade.

## Stack
- React Markdown
- Remark GFM
- CSS Custom Components
- Syntax Highlighting (e.g. `react-syntax-highlighter`)
